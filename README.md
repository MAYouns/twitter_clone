# Twitter Clone

A responsive, static Twitter (X) UI clone built with pure HTML and Tailwind CSS v4.

## Features

- **Responsive Design**: Adapts beautifully to both mobile and desktop screens.
- **Dark Mode Support**: Built-in toggle in the header to switch between light and dark themes.
- **Modern Styling**: Styled efficiently using the highly-customizable Tailwind CSS v4.
- **Core UI Components**:
  - Sidebar navigation menu.
  - Tweet composer area with action icons.
  - Scrollable main tweet feed.
  - Right sidebar with a search bar, "What's happening" section, and "Who to follow" suggestions.

## Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine to manage the Tailwind CSS dependencies and build process.

## Installation & Setup

1. **Install dependencies**:
   Run the following command in the root of the project to install Tailwind CSS:

   ```bash
   npm install
   ```

2. **Run the build process**:
   The project uses the Tailwind CLI to compile the CSS. To start the development watcher, run:

   ```bash
   npm run dev
   ```

   This command watches for changes in `./src/input.css` and `./src/index.html`, automatically compiling the generated styles to `./dist/output.css`.

3. **View the Project**:
   Simply open the `src/index.html` file in your preferred web browser to view the application. No local server is strictly required for the static HTML, but you can use an extension like VS Code's "Live Server" for a better development experience.

## Technologies Used

- [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Font Awesome 7.0](https://fontawesome.com/) (for UI icons)

## Project Structure

- `src/index.html`: The main HTML layout and content.
- `src/input.css`: The source stylesheet for Tailwind directives.
- `dist/output.css`: The compiled CSS file generated by Tailwind (referenced in the HTML).
- `assets/`: Contains the images used statically in the project.
- `package.json`: Contains project metadata and the Tailwind CLI build scripts.
